# Task 6.2: Implement Web Frontend (React + TypeScript)

## Description
Scaffold and implement the Boomerang Web Panel using React, TypeScript, and Vite. This panel will provide a user-friendly interface for managing tasks, including listing, viewing details, and registration.

## Requirements
- Use **Vite** for project scaffolding.
- Use **Shadcn/UI** (with Tailwind CSS) for the design system and components.
- Use **React Router** for client-side routing.
- Use **TanStack Query** (React Query) for state management and API integration.
- Implement **Authentication**: A login page that stores the session ID in `localStorage`.
- Implement **Dashboard Layout**: A sidebar for navigation and a header for user information.
- Implement **Task Management**:
    - **Task List**: Table view with pagination and filtering (client ID, time range).
    - **Task Details**: View detailed information about a specific task.
    - **Task Registration**: A single-page form with fields for `payload`, `delayMs`, and `repeatIntervalMs`.
- **Note**: Skip real-time updates and monitoring (Task 6.2.2) for this phase.

## Technical Details
- **Module Name**: `boomerang-web-ui` (at project root).
- **Backend Integration**: Proxy requests to the Spring Boot backend (`boomerang-web-backend`) running on its default port.
- **Styling**: Vanilla CSS or Tailwind CSS as per Shadcn/UI defaults.
