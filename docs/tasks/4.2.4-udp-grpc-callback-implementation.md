# Task 4.2.4: Implement UDP and gRPC Callback Protocols

## Objective
Implement the outbound delivery logic for UDP (fire-and-forget) and gRPC protocols.

## Requirements
- **UDP:** Netty-based datagram sender.
- **gRPC:** Managed channel and stub implementation using the service defined in Task 4.2.1.
- Global channel pooling for gRPC to avoid overhead.

## Proposed Changes
### `io.boomerang.server.callback.UdpCallbackHandler`
- Sends datagrams to the target endpoint.

### `io.boomerang.server.callback.GrpcCallbackHandler`
- Uses `BoomerangCallbackGrpc.BoomerangCallbackBlockingStub`.

## Verification Plan
- Integration tests with a mock gRPC service.
- Integration tests for UDP (verifying packet arrival).
